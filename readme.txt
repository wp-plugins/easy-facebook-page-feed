Easy way to display a customizable and responsive Facebook page feed, on your Wordpress website.

== Description ==

Easy Facebook Post Feed, allow users to display a **customizable** and **responsive** Facebook feed. It's quickly to use by widget or shortcode anyway.

= Features =
* **Customize** layout option like: color, border, number of post to display, choose to display image, width and height (px and % for responsive);
* **Widget** available;
* **Shortcode** available: [easy-facebook-post-feed];
* **Work with Facebook API** it's simple to set up: just enter your page id and Facebook API
* **Languages available**: English, Italian

How I get Facebook Api?
1. Connect at [developers.facebook.com](developers.facebook.com) to create your own app;
2. Click on "My App" menu and "Add a New App";
3. Choose app for website (www) and give it a name;
4. Copy App id and secret and paste in App Setting

Read complete documentation and make your question [here](http://www.marioconcina.it/blog/easy-facebook-post-feed-plugin-for-wordpress)

== Installation ==

1. Install the Easy Facebook Post Feed via the WordPress plugin directory, or by uploading the files to your web server;
2. Activate the plugin;
3. Navigate to the **Facebook Feed menu** to configure your feed;
4. Use widget or shortcode `[easy-facebook-post-feed]` in your post;
5. Enjoy!
